Лабораторна робота № 3 >>> Аналіз задачі, теоретичні обґрунтування вибраного методу розв’язання задачі, розрахункові формули, посилання на літературні джерела

Завдання 1:

Для виконання першого завдання: реєстрація картки читача в бібліотеці, створимо клас Reader, де опишемо поля з даними читача, а саме: ім’я, прізвище, паспорт, адреса проживання, мобільний телефон. Та створимо метод subscriptionCreation(); Цей метод буде запрошувати у користувача, щоб той увів всі дані, що є необхідними для створення читача. Після виклику цього методу та внесення всіх даних через консоль, ці дані записуються в текстовий файл "readerInfo.txt"

Завдання 2:

Для додавання книги створимо клас Library, де створимо відповідні поля та опишемо метод addBook(). Сенс цього методу полягає в тому, щоб отримати дані про книгу та записати їх в текстовий файл "library.txt". Записані ці дані будуть в один рядок у вигляді: "Назва книги" Автор - рік видання.

Завдання 3:

Для отримання книги використаємо клас Library, де опишемо метод takingBook(). Завдяки цього методу ми зможемо шукати книгу за алфавітом з відсортованого файлу "library.txt". Операція сортування відбувається за допомогою методу Sort(). Коли наші книги відсортувалися за алфавітом та перезаписалися до текстового файлу, ми можемо нарешті працювати. Якщо користувач вводить назву книги "Воно", то на консоль буде виведена відповідна книга з її характеристиками. Якщо це та книга, про яку думав читач, то він натискає "Так" і біля книги ставиться відмітка з іменем та прізвищем читача. Завдяки пошуку книги саме за символами, а не словами, ми можемо шукати книги саме за певною комбінацією літер. Наприклад, користувач вводить "Зел". На консолі буде виведена книга "Зелена миля". Тобто користувачу не потрібно вводити величезний текст. Це створено тому, що бувають книги, як з маленькими назвами (напр. "Воно"), так і з довгими (напр. "Хіба ревуть воли, як ясла повні").

Завдання 4:

Для отримання інформації про книгу опишемо метод searchBookInfo() в класі Library. Він буде шукати книгу в текстовому файлі та виводити всі дані на консоль. Цей метод використовує алгоритм лінійного пошуку, який працює за час O(1). А потім на консоль виводиться вся необхідна інформація.

Завдання 5:

Для отримання інформації про бібліотеку опишемо метод outputLibInfo() в класі Library. Цей метод буде виводити всі дані полів класу Library, а саме, назва бібліотеки, адреса та книжковий фонд. Потрібно зауважити, що, якщо дані про бібліотеку не будуть введені завчасно, то на консоль будуть виведені пусті поля.

Завдання 6:

Метод task1() знаходиться в класі Content. Цей метод є сукупністю всіх методів, які стосуються виконання блоку 1 з лабораторної роботи №2. Для читання контенту викликаємо метод task1() класу Content.

Завдання 7:

Для розрахунку відсотку зменшення ризику захворювань мозку, скористуємося формулою, поданою в методичних вказівках. Для виклику цієї функції скористаємося класом Book, що вбудований в клас Library. Для цього викличемо метод riskReduction().

Завдання 8:

Для розрахунку збільшення IQ в залежності від систематичного читання, застосуємо одну з багатьох наведених в інтернеті формул. Для виклику методу IQ скористаємося вбудованим класом Book.

Завдання 9:

Для виведення інформації про студента скористаємо методом outputIdentification() класу Identification.
